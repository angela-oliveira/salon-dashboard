# salon-dashboard

## Rodando a API

[BACK](https://github.com/angela-oliveira/back-salon)

Para rodá-la, faça:

```console
> npm install ou yarn
> npm run server ou yarn server
```

Ela responderá na porta 5000.


## Rodando o FRONT

Para rodá-lo, faça:

```console
> npm install ou yarn
> npm start ou yarn start
```

Ela responderá na porta 3000.
